# 세련된 디자인의 To Do List 웹서비스 개발

본인이 원하는 디자인의 **Todo** 웹 서비스를 제작하는 프로젝트입니다.

## 평가 항목

- 의도하는 웹페이지의 레이아웃 및 스타일 구현 능력 ✔️

### 필수 구현 항목

  1. 기본적인 To Do의 CRUD(추가, 조회, 수정, 삭제) 구현 ✔️
 
  2. 복수의 메뉴를 가진 사이드바 ✔️
 
  3. 복수의 페이지와 페이지 이동 기능 ✔️
  
  4. 화면 위로 뜨는 모달창 구현 ✔️
