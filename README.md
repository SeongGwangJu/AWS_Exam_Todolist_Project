학습한 **HTML, CSS, JavaScript** 능력을 평가(시험)하기 위해

본인이 원하는 디자인의 **Todo** 웹 서비스를 제작하는 프로젝트입니다.

## 평가 기준

- 레이아웃 및 스타일 구현 능력 ✔️

#### 필수 구현 기능

  - 기본적인 To Do의 CRUD(추가, 조회, 수정, 삭제) 구현 ✔️
 
  - 복수의 메뉴를 가진 사이드바 ✔️
 
  - 복수의 페이지와 페이지 이동 기능 ✔️
  
  - 모달창 구현 ✔️

#### 추가 구현 기능(가산점)

  - 캘린더 추가 ✔️

  - 캘린더를 통한 날짜별 조회 ✔️

  - todo 상태(완료/미완료)에 따른 조회 및 상태 표시 ✔️
    
  - 완료된 일 전체 삭제 기능 ✔️
